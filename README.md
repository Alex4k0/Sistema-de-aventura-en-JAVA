# Adventure System – Mini sistema de aventura en consola (Java)

Descripción:
Proyecto académico en Java que simula un sistema de aventura por consola. El jugador elige una clase de héroe y puede ejecutar comandos de combate o recoger objetos. El programa procesa acciones mediante texto y mantiene un registro de la aventura y un inventario limitado.

Características principales:

Elección de clase de héroe mediante enum (Warrior, Mage, Archer).

Sistema de comandos con formato (Acción:Objetivo:Elemento).

Registro de acciones en un adventure log.

Inventario de 5 espacios con control de límite.

Procesamiento de comandos de ataque y de loot.

Interacción mediante consola usando Scanner.

Tecnologías utilizadas:
Java
Programación orientada a objetos básica
Aplicación de consola

Estructura del proyecto:
AdventureSystem.java

Cómo ejecutar el proyecto:

Compilar el archivo con javac AdventureSystem.java

Ejecutar con java AdventureSystem

Objetivo del proyecto:
Practicar entrada de datos, manejo de cadenas, arrays, enums, lógica condicional y estructura básica de un sistema interactivo en Java.

Mejoras futuras:
